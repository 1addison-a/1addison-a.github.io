Website Evaluations

Dropbox

A clean, user-friendly interface for file storage and collaboration.

  • Contrast: High contrast between text and background enhances readability.
  • Repetition: Consistent use of colors, fonts, and button styles across pages.
  • Alignment: Elements are neatly aligned, contributing to a structured layout.
  • Proximity: Related items, like navigation links and content, are grouped together, aiding user navigation.

Ling's Cars

A chaotic and cluttered design offering car leasing services.

  • Contrast: Excessive use of bright colors and mismatched fonts reduces readability.
  • Repetition: Inconsistent design elements create a disjointed experience.
  • Alignment: Elements are haphazardly placed, leading to a cluttered appearance.
  • Proximity: Unrelated items are placed too close together, confusing users.
